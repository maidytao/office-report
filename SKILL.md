# Office Report

Use this skill to turn the user's notes, outlines, rough drafts, or source materials into formal Chinese office documents.

## Core Principles

- Write in a formal, steady, clear, and well-organized style.
- Preserve the user's core meaning, facts, names, dates, numbers, positions, and conclusions.
- Do not invent facts, policies, statistics, responsible units, approval opinions, or implementation progress.
- If key facts are missing, use neutral placeholders or ask concise follow-up questions when the missing information affects accuracy.
- Prefer practical official-writing language over ornate rhetoric.
- Make the document suitable for submission to leaders, government departments, or internal company circulation.

## Workflow

1. Determine the material type first.
   - Identify whether the task is a government report, meeting minutes, feasibility study report, work summary, situation explanation, notice, announcement, or another formal document.
   - If the type is unclear, infer from the user's wording, audience, and intended use. State the assumed type briefly when helpful.

2. Organize the title and structure.
   - Create a concise, formal title matching the document type.
   - Choose a conventional structure for the type.
   - Keep headings parallel, short, and logically ordered.
   - Use Chinese document numbering consistently: `一、`, `（一）`, `1.`, `（1）` as needed.

3. Preserve the user's core meaning.
   - Keep all provided facts and conclusions aligned with the source.
   - Rephrase rough language into formal written Chinese without changing intent.
   - Remove repetition, casual phrasing, and ambiguous expressions.
   - Do not add unsupported achievements, causes, risks, policy bases, or commitments.

4. Add transitions and logical connections.
   - Add necessary connecting phrases to clarify background, current status, problems, analysis, measures, and next steps.
   - Strengthen cause-and-effect, timeline, responsibility, and priority relationships.
   - Keep transitions restrained and natural.

5. Final check.
   - Check title, recipient line if present, body structure, numbering, paragraph order, and closing language.
   - Check typos, grammar issues, punctuation, duplicated words, and inconsistent terms.
   - Check whether the tone is appropriately formal and whether unsupported facts were introduced.

## Document Type Guidance

### Government Report / Work Report

Use a structure such as:

- Background or basic situation
- Main work completed
- Existing problems or difficulties
- Cause analysis when needed
- Next-step work arrangements or requests for support

Use clear phrases such as `现将有关情况报告如下：`, `下一步，将重点做好以下工作：`, and `恳请予以指导支持。`

### Meeting Minutes

Use a structure such as:

- Meeting time, location, participants, host, and recorder when provided
- Meeting topic
- Main discussion points
- Decisions or agreed matters
- Follow-up tasks, responsible units, and deadlines when provided

Do not fabricate attendees, decisions, responsibilities, or deadlines.

### Feasibility Study Report

Use a structure such as:

- Project background and necessity
- Current conditions or basis
- Construction/content plan or implementation approach
- Feasibility analysis
- Risk factors and safeguards
- Investment, schedule, and expected benefits when provided
- Conclusion and recommendations

If data is missing, mark it as `待补充` rather than inventing figures.

### Work Summary

Use a structure such as:

- Overall situation
- Main work and results
- Experience and practices
- Problems and deficiencies
- Next-step plan

Keep achievements proportional to the user's provided material.

### Situation Explanation

Use a structure such as:

- Basic situation
- Cause or process
- Current handling status
- Impact assessment when provided
- Corrective measures or next steps

Use objective, responsible language. Avoid emotional or defensive phrasing.

### Notice / Announcement

Use a structure such as:

- Purpose and basis
- Main matters
- Requirements for implementation
- Time, scope, contacts, or submission method when provided
- Closing formula

Keep instructions direct, precise, and easy to execute.

## Output Rules

- Provide the polished document directly unless the user asks for analysis first.
- If assumptions were necessary, briefly list them before or after the document.
- If source material is too thin for a complete document, produce a usable framework with `待补充` placeholders.
- Keep language concise and avoid excessive slogans.
- Prefer simplified Chinese punctuation and formatting.
